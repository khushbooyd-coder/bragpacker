# Bragpacker WordPress Plugins

Custom plugins built for bragpacker.com

## Plugins

### bragpacker-cart-ux (v1.4.0)
Split checkout (Buy vs Rental), mixed cart blocking with notices, correct checkout routing, Buy Now (AJAX + non-AJAX), side-cart routing, qty steppers, rental checkout UX, refundable deposit line.

### bragpacker-kh-buy-now (v1.1.0)
Buy Now button for Bragpacker Buy Store (PDP + archive). Intercepts Add to Cart and Buy Now clicks on archive pages to block mixed carts with notices. Keeps Add to Cart visible on PDP.

### bragpacker-rental-checkout (v6.0)
Rental checkout only — City dropdown, Metro grouping (Mumbai MMR, Pune MMR, Delhi NCR), pincode auto-detection, popup for Other areas, backend validation.
